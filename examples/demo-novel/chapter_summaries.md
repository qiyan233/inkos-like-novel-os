# Chapter Summaries

## Chapter 1 - 第一章 库房旧玉
- POV: 林烬
- Summary: 林烬在徐家旧库房清点旧物时，察觉家传玉佩的手感与磨痕不对，第一次生出“有人提前动过它”的怀疑。
- Key events:
  - 林烬在旧库房发现玉佩异常。
  - 徐安出现，但没有正面回应“谁先碰过玉佩”。
- State changes:
  - 林烬确认玉佩并非自己记忆中的旧物状态。
- Hooks opened:
  - 是谁在林烬之前碰过玉佩？
- Hooks advanced:
  - 徐安的沉默让怀疑第一次有了明确对象。
- New facts:
  - 玉佩边角过于完整，不像多年旧物。
- Character knowledge changes:
  - 林烬知道玉佩异常；徐安知道林烬已经注意到异常。

## Chapter 2 - 第二章 雨夜试探
- POV: 林烬
- Summary: 林烬在雨夜偏厅借旧库房记录与徐安周旋，确认对方在“最早经手人”的问题上有意回避，但仍无法直接坐实调包者身份。
- Key events:
  - 林烬故意询问记录册去向。
  - 徐安承认玉佩入库前曾短暂离手，却不肯说出名字。
- State changes:
  - 林烬确认徐安知道更多内情。
- Hooks opened:
  - 为什么徐安不肯说出最早经手人的名字？
- Hooks advanced:
  - “谁先碰过玉佩”从模糊异常变成可追查的转手顺序问题。
- New facts:
  - 玉佩在登记入库前曾经短暂脱离原保管链。
- Character knowledge changes:
  - 林烬知道问题不只在库房内部；徐安知道林烬不会轻易放过这条线。
