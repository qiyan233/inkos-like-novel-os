# Book Rules — 玉佩疑云

## Protagonist lock
- personality: 林烬谨慎、善于观察，不会在证据不足时直接摊牌。
- default response under pressure: 先试探，再确认。
- moral floor / ceiling: 不会主动栽赃，也不会无凭无据毁掉盟友。

## Behavioral constraints
- 林烬的怀疑必须来自可见细节，而不是作者视角直给答案。
- 徐安的反应应当含混可疑，但不能直接承认。

## Prohibitions
- 不允许全知旁白直接公布幕后人。
- 不允许角色突然知道自己尚未获取的信息。

## Tone goals
- 克制、带压迫感，少解释，多让细节说话。

## Pacing rules
- 每章至少推进一个 hook。
- 前两章只推进怀疑，不回收核心悬念。
