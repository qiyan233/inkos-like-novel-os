## Operating rule

Before writing or revising, read:

- story_bible.md
- book_rules.md
- outline.md
- current_state.md
- latest entries in chapter_summaries.md
- pending_hooks.md
- character_matrix.md
- character_knowledge.md (if maintained)

## Directories

- `chapters/` — chapter drafts/finals
- `reviews/` — audit and revision reports
- `scratch/` — optional planning or temporary notes
